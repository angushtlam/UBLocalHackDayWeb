Starter Packs

Here are some downloadable starter packs. These are working projects that can show you how to quickly get started in creating your very own project! We've added some extra resources for each of these packs to help you understand it more.

If you need help with anyone of these, please call over a mentor! Remember you don't need to use these, you are more than welcome to make whatever you want!

Bootstrap Easy

Bootstrap is a CSS and Javascript framework designed to speed up creating a website. You can create beautiful websites in lightning speed, perfect to use for your portfolio.

We recommend you to learn this if you have a basic level of understanding for HTML and CSS.

Documentation

Flask Easy

Flask is a Python webserver microframework for creating simple webapps. It's perfect for creating most web-based services!

We recommend you to learn this if you have a basic level of understanding for Python or other scripting languages.

GitHub Source of Starter Pack
Documentation

p5.js Medium

p5.js a Javascript client-side library (for browsers etc.) for creating graphic and interactive experiences. This allows you to create amazing 2D animated and interact with them with a few lines of code!

We recommend you to learn this if you have a basic level of understanding for Javascript or other scripting languages.

GitHub Source of Starter Pack
Documentation

React Hard

React is a Javascript library designed to handle the visuals (or the "View" of the MVC model) of a webapp. It's essential in keeping front-end code organized, by creating and reusing "Components".

We recommend you to learn this if you have an intermediate level of understanding for Javascript, and have experience with the command line.
